Log Output

The Log Output at the bottom allows to display the log and filter it for relevant information.

You can

  1. Hide (arrow-down) the Output.
  2. Enable/Disable auto-scroll for the displayed messages.

  3. Filter which message type should be displayed. There are four types available:

    • : Debug information generated by INSTINCT.
    • : Information, e.g. elapsed time for task.
    • : Warnings the user should address for better operation.
    • : Errors that had been encountered (may cause INSTINCT not to run a Flow).

    The menu offers four modes:

    • debug: Shows debug and all below types (shows all).
    • info: Shows info and all below types (default).
    • warning: Shows warnings and all below types.
    • error: Shows (only) errors.
  4. Search bar to filter for a specific output manually.
  5. Display area for the (filtered) output.
