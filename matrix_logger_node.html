Introduction

The MatrixLogger Node logs (saves) a Matrix.

Connections

  • Input : The Node has a Matrix-flow input (see Demo Node).
  • Output : The Node has no output connection: In the Settings the output file can be selected.

Settings

The Settings offer to enter a file path or choose one to save the csv-file generated.

Example Flow

If you don't know how to reproduce any of the explained steps, read the Getting Started first.

Required Nodes:

  • Demo Node (Demo Node)
  • MatrixLogger Node

Steps

  1. Add the Demo and MatrixLogger Node to your workspace.
  2. Connect the Matrix-output of the Demo Node with the MatrixLogger.
  3. Open the Demo Node Settings:
    • Change the output matrix as you wish.
  4. Open the MatrixLogger Node Settings:
    • Select a path to save the file.
  5. Run the Flow, compare the resulting file with your input.
