Introduction

The RtklibPosFile Node reads RTKLIB .pos-files into INSTINCT.

Connections

  • Input : The Node has no input connection: In the Settings the input file can be selected.
  • Output : The Node has an RtklibPosObs-Flow output.

Settings

The Settings offer to enter a file path or choose a .pos-file with the file browser.

The Node should initialize as soon as a file in selected, if not, it can be done manually. This way, the Node displays the found entries from the read file.
